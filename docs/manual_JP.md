# EventSequencer 操作マニュアル (v0.9.1)

EventSequencer は、Next.js と Electron をベースにしたプロフェッショナル向けのスタンドアロン DAW シーケンサーです。

## 1. 基本操作

### トランスポート制御
- **PLAY**: シーケンスを開始します。
- **PAUSE**: 一時停止します。
- **STOP**: 停止し、再生位置を 00:00:00 にリセットします。
- **SEEK**: 上部のタイム表示をダブルクリックして数値を入力するか、タイムライン上部のルーラーをドラッグして再生位置を変更できます。

### タイムライン操作
- **ズーム**: `Ctrl + マウスホイール` でタイムラインの拡大・縮小が可能です。
- **追従 (Chase)**: 再生中に画面を自動でスクロールさせます。手動でスクロールしたりシークしたりすると自動的にオフになります。
- **ループ範囲**: ルーラー上で `Shift + ドラッグ` することで、特定の範囲をループ再生できます。解除はルーラー上の右クリックまたはステータスバーの「CLEAR」ボタンで行います。

## 2. トラックとデバイスの管理

### サイドバー（TRACK LIST）
- **デバイス追加**: 下部の「Add Device」ボタンで新しいデバイスを追加できます。
- **トラック追加**: デバイス名の横にある「＋」ボタンでトラックを追加します。
- **名前の変更**: デバイス名やトラック名をダブルクリックして編集できます。
- **並び替え**: デバイスやトラックをドラッグ＆ドロップして自由に順番を入れ替えられます。
- **自動幅調整**: **「TRACK LIST」というヘッダー文字をダブルクリック** すると、名前の長さに合わせてサイドバーの幅が自動的に調整されます。

### イベント操作
- **イベント作成**: トラック上の空白部分をダブルクリックしてイベントを作成します。
- **移動**: イベントをドラッグして時間を移動できます。
- **時間の直接編集**: イベントを選択し、右側のプロパティパネルで **Execution Time** を直接入力して、ミリ秒単位での精密な時間調整が可能です（`HH:MM:SS.mmm` 形式）。
- **複製**: `Alt + ドラッグ` でイベントをコピーできます。
- **削除**: イベントを選択して `Delete` キーまたはプロパティパネルから削除します。

## 3. ネットワーク制御 (TCP / OSC)

### TCP リモートコントロール
外部機器やソフトウェア（Stream Deck など）からネットワーク経由で制御可能です。
- **ポート**: `9001` (TCP)
- **コマンド**:
  - `@play`: 再生開始
  - `@pause`: 一時停止
  - `@stop`: 停止（リセット）
  - `@status`: 現在の状態取得 (`@play@`, `@pause@`, `@stop@` が返ります)

### OSC サポート (受信)
外部からの OSC メッセージによる操作も可能です。
- **ポート**: `8000` (UDP)
- **トランスポート操作**:
  - `/transport/play`: 再生
  - `/transport/stop`: 停止
  - `/transport/pause`: 一時停止

## 4. イベント詳細と OSC 型指定

### RAMP イベント
一定時間かけて値を変化させるオートメーションイベントです。
- **高精度エンジン**: OSC 通信において、開始値と終了値を確実に送信するエンジンを搭載しており、フェーダー操作などの精密な制御に適しています。
- **OSC 型プレフィックス**: Command Template 内で以下のプレフィックスを使用することで、OSC のデータ型を明示的に指定できます。
  - `i:` : Integer (32bit 整数)
  - `f:` : Float (32bit 浮動小数点)
  - `d:` : Double (64bit 浮動小数点) - **高精度フェーダー操作に推奨**
  - `s:` : String (文字列)
- **使用例**: `VOL d:{value}` と設定すると、高精度な 64bit Double 型で送信されます。プレフィックスを省略した場合は自動的に型が推論されます。

## 4. セキュリティと設定

- **LOCK モード**: 右上の鍵アイコンをクリックすると「LOCK モード」になります。本番演奏中に誤ってイベントを動かしたり、設定を変更したりするのを防ぎます。
- **スタンドアロン実行**: `dist/EventSequencer-win32-x64/EventSequencer.exe` を実行するだけで動作します。Node.js のインストールは不要です。
